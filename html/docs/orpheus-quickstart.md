# Orpheus Quick Start Guide

Get your Orpheus unit up and running in minutes.

---

## 1. Unbox Your Unit

Verify you have all components:
- Orpheus main unit
- LiFePO4 battery (if separate)
- Solar panel connectors (MC4)
- Speaker or speaker cables
- This quick start guide

## 2. Connect Power

### Battery
If not pre-installed, connect the battery observing correct polarity.

### Solar Panel
Connect your solar panel using the MC4 connectors on the enclosure.

## 3. Connect Speaker

Attach your external speaker to the audio output terminals on the enclosure.

## 4. Power On

1. Press the power button
2. Wait ~6 seconds for initialization to complete
3. The main interface will appear

> **Note:** The brief initialization period is normal - the system is loading the database and syncing power data.

## 5. Configure Location (For Astral Features)

1. Go to **Settings > Astral Setup**
2. Enter your GPS coordinates
3. Set your timezone
4. Save settings

## 6. Test Audio

1. Select **Manual** mode
2. Tap **Load** and choose an audio file
3. Tap **Play**
4. Adjust volume as needed

## 7. Create Your First Schedule

1. Select **Interval** mode
2. Tap **Add Schedule**
3. Set your desired times
4. Select audio file(s)
5. Save and enable the schedule

---

## You're Ready!

Your Orpheus unit is now configured for basic operation.
